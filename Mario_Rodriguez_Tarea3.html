Ejercicio 2

A partir del sistema de ecuaciones diferenciales anterior, discretizamos el problema tomando un entero $N \geq 1$ y definiendo los saltos en el tiempo como: $$\Delta{t} = \frac{T}{N}$$ Y los niveles de tiempo como: $$t_{n} = n\Delta{t}$$ para todo $n = 0, 1, 2, ..., N \quad$ y $\quad t_{0} = 0 \quad$ y $ \quad t_{N} = T$.

Establecemos que $F_{n}$ es una aproximacin de $F(t_{n})$

Para derivar una aproxiamcin de la solucin de $$F'(t)=(2-S(t))F(t)$$ Se usa la serie de Taylor de manera que: $$F(t + \Delta{t}) = F(t) + \Delta{t}F'(t) + O(\Delta{t^2})$$ Despejando $F'(t)$: $$F'(t) = \frac{F(t + \Delta{t}) - F(t)}{\Delta{t}} + O(\Delta{t})$$ Si se establece esta observacin para el momento en el que $t = t_{n}$ y puesto que se busca establecer un $\Delta{t} \approx 0$ se espera que $O(\Delta{t}) \approx 0$ por lo que: $$F'(t_{n}) = \frac{F(t_{n+1}) - F(t_{n})}{\Delta{t}}$$ Y dado que $F_{n} \approx F(t_{n})$: $$F'_{n} = \frac{F_{n+1} - F_{n}}{\Delta{t}}$$ Y despejando $F_{n+1}$ $$F_{n+1} = F_{n} + \Delta{t}F'_{n}$$ Puesto que $F'(t) = (2-S(t))F(t)$: $$F_{n+1} = F_{n} + \Delta{t}(2-S_{n})F_{n}$$

Siguiendo el mismo razonamiento obtenemos la aproximacin numrica de $S'(t)$: $$S_{n+1} = S_{n} + \Delta{t}(F_{n} - 1)S_{n}$$

def aproximacion_numerica(f0, s0, dt, T):
    N = int(T / dt)

    F = np.zeros(N + 1)
    S = np.zeros(N + 1)

    F[0] = f0
    S[0] = s0

    for n in range(N):
        F[n + 1] = F[n] + dt * (2 - S[n]) * F[n]
        S[n + 1] = S[n] + dt * (F[n] - 1) * S[n]

    return F, S

El esquema Clank-Nicolson es de la siguiente forma: $$ \frac{U_{n+1} - U_{n}}{\Delta{t}} = \frac{1}{2}[f(U_{n+1}) + f(U_{n})] \qquad donde \qquad U'(t) = f(U(t)) $$

Sustituyendo por $ F'(t) \hspace{0.2cm} y \hspace{0.2cm} S'(t) $

$$ \frac{F_{n+1} - F_{n}}{\Delta{t}} = \frac{1}{2}[(2-S_{n})F_{n} + (2-S_{n+1})F_{n+1}] $$ $$ \frac{S_{n+1} - S_{n}}{\Delta{t}} = \frac{1}{2}[(F_{n}-1)S_{n} + (F_{n+1} - 1)S_{n+1}] $$

Resolviendo para $F_{n+1} \quad y \quad S_{n+1}$: $$ F_{n+1} = \frac{F_{n}\left(1 + \Delta{t} - \frac{\Delta{t}}{2}S_{n} \right)}{\left(1 - \Delta{t} + \frac{\Delta{t}}{2}S_{n+1} \right)} $$ Y $$ S_{n+1} = \frac{\left(1 - \frac{\Delta{t}}{2} + \frac{\Delta{t}}{2}F \right)} {\left(1 + \frac{\Delta{t}}{2} - \frac{\Delta{t}}{2}F_n+1 \right)} $$

# Y se aproximan los valores del sistema lotka-volterra
# para un periodo de tiempo de t=0 hasta t=10
def clank_nicolson(f0, s0, delta_t, T, soluciones):
    N = int(T / delta_t)
    F = np.zeros(N + 1)
    S = np.zeros(N + 1)

    F[0] = f0
    S[0] = s0

    for n in range(N):
        # Definir los valores de F_n, S_n y dt
        valores = {F_n: F[n], S_n: S[n], dt: delta_t}

        # Evaluar las soluciones con estos valores
        soluciones_evaluadas = [sol.subs(valores) for sol in soluciones]

        F[n + 1] =  soluciones_evaluadas[1][0]
        S[n + 1] =  soluciones_evaluadas[1][1]

    return F, S

Ejercicio 6

Una aplicacin interesante de este sistema de ecuaciones diferenciales est relacionado con la economa, especficamente con la relacin productor-consumidor, y como estos interactuan en el mercado.

Dentro de esta aplicacin, el papel del depredador lo toma el productor, el cual depende de la existencia del consumidor para poder generar ingresos. Cuando el nmero de consumidores es alto, los productores pueden proliferar, es decir, la demanda generar incentivos para que ms personas quieran producir un bien o servicio y tomar una porcin del mercado mientras que, si los consumidores disminuyen, habr muchos productores que preferirn detener su produccin pues no habr mercado suficiente para generar ganancias.

Por su parte, los consumidores toman el papel de presa en este modelo. Si los productores incurren en prcticas que desincentivan el consumo, entonces los consumidores "disminuirn", es decir, habr menos personas dispuestas a adquirir los bienes y servicios de los productores.

La dinmica econmica que describe este modelo es la siguiente:

En una primera fase, los productores son pocos, y los consumidores muchos, as que los productores empiezan a "consumir" a estos consumidores, es decir, les venden sus productos. Este exceso de demanda, genera que haya un crecimiento en la cantidad de productores.

Cuando el mercado se empieza a saturar de productores, representa una menor cantidad de consumidores que puede captar cada consumidor, as que algunos productores empiezan a "morir de hambre", lo cual representa una disminucin de los productores. Adems, los consumidores pueden percibir una saturacin en el mercado, lo cual puede llevar a la pdirda de inters y por lo tanto, la disminucin de consumidores de ese producto.

Al disminuir la saturacin en el mercado, el ciclo puede volver a iniciar, creando una nueva etapa de crecimiento de la poblacin de productores.
